# cash-register

> 使用Electron + Vue + Element UI + NeDB搭建的小商店记账软件

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


```
